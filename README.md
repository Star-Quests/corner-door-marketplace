# CORNER DOOR - Anonymous Crypto Marketplace

A simple, clean, anonymous crypto-only marketplace supporting BTC, ETH, and SOL payments.

## Features

- Anonymous user accounts (username + password only)
- No emails, phone numbers, or personal information
- USD pricing with real-time crypto conversion
- BTC, ETH, SOL payments with QR codes
- Real-time notifications system
- Product delivery system (PDF, Word, Videos, Images)
- Admin-controlled star ratings
- User rating system (admin controlled)
- Wallet management with delete functionality
- Simple admin panel
- No escrow system
- Lightweight and fast
- Clean, dark UI design
